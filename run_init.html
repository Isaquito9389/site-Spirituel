Initialisation des images pour le site

Cliquez sur le bouton ci-dessous pour initialiser les catégories d'images nécessaires pour la page d'accueil.

Cette opération va:

  • Vérifier si la table image_library existe et la créer si nécessaire
  • Ajouter les catégories d'images manquantes avec des chemins par défaut
Initialiser les images

Une fois l'initialisation terminée, vous pourrez utiliser la bibliothèque d'images dans l'administration pour gérer les images du site.
